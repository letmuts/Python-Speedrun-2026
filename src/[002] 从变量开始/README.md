# 何为变量

计算机是用来计算的，计算的输入端需要输入值，而计算机需要记录我们输入的值，然后才能计算

变量就是用于存储值的

计算机存储值的方式是给每个值一个唯一的 ID 或者叫它身份证号，ID 和一个随机的数字没有区别，人类无法记忆，所以我们需要给每个值起个名字，这就是变量名

还记得前面文章里创建的 `1.py` 吗？用 VSCode 打开它并清除文字

```python
a = 123
b = "abc"
print(a, b)
```

将这段代码复制粘贴过去，按 `F5` 运行，它输出了 `123 abc`

`123` 和 `"abc"` 是两个值，我们不关心这两个值到底存在计算机的哪里，我们只关心这两个值在 Python 代码里的变量名叫 `a` 和 `b`，Python 解释器会根据我们起的名字找到对应的值的位置

# 命名规则

为了保证变量名与其他符号之间不冲突不混淆，Python 强制变量名只允许由字母、数字、下划线组成，以及允许非 ASCII 字符例如中文

```python
a_1 = 1  # 正确
_b_2 = 2  # 正确
3_c = 3  # 错误，因为数字不能放在开头
变量_4 = 4  # 正确
```

复制这段代码到 `1.py`

* `3_c` 这里变红报错了，这就是 Pylance 的静态检查的作用，能在运行前发现错误
* `#` 后的字符变绿了，因为 `#` 是注释符号，注释里的内容不会被 Python 解释器执行，可以用来写一些解释代码的注释

# 赋值

`a = 123` 这里的 `=` 并不是判断 `a` 是否等于 `123`，它的意思是将变量 `a` 的位置改到值 `123`，也就是 Python 解释器会根据 `a` 变量找到值 `123` 在计算机内的位置

既然 `变量` 是会变化的量，那 `a` 的位置肯定也不会是固定的，我们可以修改任何一个变量所指向的位置

```python
a = 123
a = 456
```

这段代码里，`a` 先是指向值 `123` 所在的位置，然后改为了指向值 `456` 值所在的位置

修改变量指向的值的位置的行为，称为 `赋值`，也就是将值赋予变量

# 类型

每个值都是有类型的，就如前篇文件说的，可以在官方文档中查看内置类型 <https://docs.python.org/zh-cn/3/library/stdtypes.html>

```python
var_1: int = 123  # 整数
var_2: float = 123.45  # 浮点数
var_3: str = "abc123"  # 字符串
var_4: str = 456  # Pylance 在这里报错
```

复制这段代码到编辑器，这里我们使用了 3 种内置类型，也是最常用的三种类型

浮点数就是小数，字符串可以存储任何字符

`var_1: int` 这里的 `:int` 是 Python 的类型标注语法，类型标注能让静态检查器在运行前检查出错误，例如 `var_4` 这里 Pylance 就报错了

**Python 的类型是不可变的**，我们只能改变变量的指向，不能改变值的类型（甚至不能改变值本身，后面讲对象时会解释这点），有些看似改变类型的操作，实际上是生成新的值

```python
a = 1 * 1.0
b = 1 * 1
print(type(a), type(b))  # <class 'float'> <class 'int'>
```

# 运算符

同数学一样，有运算符才代表需要计算

还记得前篇文章提到的 REPL 吗？在 VSCode 中按 `Ctrl + ~` 打开内置终端，输入 `python` 直接启动，我们来输入简单的计算

```python
>>> 1 + 2  # 加
3
>>> 3 - 4  # 减
-1
>>> 5 * 6  # 乘
30
>>> 7 / 8  # 除
0.875
>>> 12 // 10  # 除（取整）
1
>>> 12 % 10  # 取模（也就是取除法的余数）
2
>>> 2 * (3 + 1)  # 括号内优先计算
8
>>> 2 / (2 + (-3 + -5) * 4)  # 括号可以嵌套，从输出可见浮点数 float 的精度不是无限的，所以高精度计算需要使用整数
-0.06666666666666667
>>> 3 ** 4  # 幂（次数、指数）
81
>>> 4 ** 0.5, 16 ** 0.25  # 小数幂就是开方，与数学一致
(2.0, 2.0)
```

不难发现很多计算会导致整数 int 变为浮点数 float，需要注意类型匹配
